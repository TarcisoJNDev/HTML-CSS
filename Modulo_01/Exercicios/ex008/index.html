Principais formatações

Negrito / Destaque

Nessa frase temos um termo em negrito usando a tag <b> (Não SEMÂNTICA)

Nesta frase, tenho um Termo em destaque usando a tag <strong> (SEMÂNTICA)

Itálico / Ênfase

Nesta frase, temos um termo em itálico usando a tag <I> (NÃO SEMÂNTICA)

Nesta frase, temos um termo em ênfase usando a tag <EM> (SEMÂNTICA)

Texto Sublinhado

Podemos marcar um texto como Sublinhado utilizando a tag <ins> (SEMÂNTICA)

Existe também o sublinhado com a tag u (NÃO SEMÂNTICA)

Texto marcado

Podemos criar também um texto marcado usando a tag <mark>

Podemos criar também um texto marcado com cor diferente usando a tag <mark style="">
(Não utlizado, para mudar a cor de algo é dentro do CSS)

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno nesse parágrafo.

Texto deletado

Podemos marcar um texto como excluído para indicar que deve ser lido, mas não considerado. utilizando a tag <del>

Texto sobrescrito

Para inserir coisas do tipo x20+3 onde a palavra/numero fica em cima do texto. Utiliza a tag <sup>

Texto subescrito

Para inserir coisas do tipo H2O onde a palavra/numero fica em baixo do texto. Utiliza a tag <sub>
